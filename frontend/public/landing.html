Why ClinIQ

Not a chatbot. An agent.

🔗
Always cited
Every claim links to a real PubMed paper. The agent cannot generate an answer without retrieved context — hallucination is structurally blocked.
🔄
Self-correcting
The reflect node critiques its own output. If the answer is off-topic or ungrounded, the agent retries automatically — up to 3 loops.
📊
RAGAS evaluated
Faithfulness, answer relevancy, and context recall are measured on every response using the RAGAS framework — not just vibes.
